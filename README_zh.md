# 网页列表导出与自动截图工具 (List Exporter & Auto Screenshotter)

![预览](./assets/preview.png)

[English](./README.md) | [简体中文](./README_zh.md)

一款功能强大的 Chrome 浏览器扩展，旨在自动化从任何网页中提取列表数据并捕捉每个条目截图的过程。非常适合数据采集、UI/UX 竞品分析和内容归档。

## 🚀 功能特点

- **🔍 自动扫描**: 启发式检测网页上的重复元素（列表、商品网格、新闻信息流等）。
- **🎯 智能高亮**: 在导出前提供检测到的列表条目的实时视觉反馈，确保准确性。
- **📸 自动截图**: 自动遍历所选列表，将每个条目滚动到视图中，并使用 `html2canvas` 进行精准截图。
- **📊 深度数据提取**: 不仅仅提取文本，还支持捕捉：
    - 可见文本内容
    - 图片 URL 和 Alt 文本
    - 链接 (Href) 和标题
    - 自定义数据属性 (`data-*`)
    - ARIA 标签和描述
    - 表单输入值
- **📦 ZIP 导出**: 将所有捕捉到的图片和完整的元数据 CSV 打包成一个整洁的 ZIP 文件。
- **⚡ 仅数据模式**: 只需要数据而不需要图片？跳过截图步骤，实现闪电般的 CSV 导出。

## 🛠️ 安装方法

1. 克隆或下载本仓库。
2. 打开 Chrome 浏览器并访问 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，然后选择本项目文件夹。

## 📖 使用指南

1. 访问包含列表或网格的任一网页（例如：电商网站、新闻列表或搜索结果页）。
2. 点击工具栏中的 **List Exporter** 扩展图标。
3. 点击 **"Scan for Lists"（扫描列表）**。扩展将识别页面上的潜在列表。
4. 在弹出窗口中点击检测到的列表，页面上会**高亮**显示对应内容以供确认。
5. （可选）如果只需要 CSV，请勾选 **"仅导出数据（无截图）"**。
6. 点击 **"Export Selected"（导出选中项）**。
7. 稍等片刻，扩展会自动滚动并处理每个条目。处理完成后，将自动下载一个 ZIP 文件。

## 🏗️ 技术栈

- **Vanilla JS & CSS**: 扩展核心逻辑和 UI。
- **[html2canvas](https://html2canvas.hertzen.com/)**: 用于将网页元素捕捉为图片。
- **[JSZip](https://stuk.github.io/jszip/)**: 用于将截图和数据打包成 ZIP 文件。
